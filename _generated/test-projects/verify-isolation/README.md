# verify-isolation

## Overview
Healthcare fullstack application built with modern technologies.

## Technology Stack
- **Frontend**: nextjs
- **Backend**: django
- **Database**: mongodb
- **Authentication**: auth0
- **Deployment**: aws

## Features
- access_control
- appointment_scheduling
- audit_logging
- data_backup
- encryption
- medical_records
- patient_portal
- secure_messaging
- telehealth

## Compliance
- HIPAA
- SOX

## Quick Start

1. Install dependencies:
   ```bash
   make setup
   ```

2. Start development environment:
   ```bash
   make dev
   ```

3. Run tests:
   ```bash
   make test
   ```
