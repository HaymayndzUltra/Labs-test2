# frontmatter-demo

## Overview
Finance fullstack application built with modern technologies.

## Technology Stack
- **Frontend**: nextjs
- **Backend**: django
- **Database**: postgres
- **Authentication**: auth0
- **Deployment**: aws

## Features
- account_management
- audit_logging
- audit_trails
- encryption
- fraud_detection
- regulatory_reporting
- reporting
- risk_assessment
- transaction_processing

## Compliance
- SOX
- PCI

## Quick Start

1. Install dependencies:
   ```bash
   make setup
   ```

2. Start development environment:
   ```bash
   make dev
   ```

3. Run tests:
   ```bash
   make test
   ```
